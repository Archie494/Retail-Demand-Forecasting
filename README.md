# Retail Demand Forecasting

This project builds an end-to-end machine learning pipeline to predict retail demand using historical sales data. The workflow includes exploratory data analysis, preprocessing, feature engineering, model training, evaluation, and business insight generation.

## Project Objective
The goal of this project was to forecast retail demand using variables related to sales, pricing, inventory, promotion, and seasonality.

## Workflow
- data cleaning and preprocessing
- exploratory data analysis
- feature engineering
- model training and comparison
- model evaluation
- business insight generation

## Models Used
- Linear Regression
- Random Forest Regressor
- XGBoost Regressor

## Best Model
XGBoost achieved the strongest overall performance and was selected as the final model.

## Key Insights
Important demand drivers included factors related to sales volume, inventory, pricing, and promotions.
