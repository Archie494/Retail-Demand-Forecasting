# Model Performance Summary

## Evaluated Models

### Linear Regression
- baseline regression model for demand prediction

### Random Forest Regressor
- improved nonlinear modeling compared to baseline regression

### XGBoost Regressor
- best-performing model in the project

## Final Outcome
XGBoost delivered the strongest forecasting performance and was selected as the final model for retail demand prediction.

## Notes
This project focused on demand prediction using sales, pricing, inventory, promotion, and seasonal features.
